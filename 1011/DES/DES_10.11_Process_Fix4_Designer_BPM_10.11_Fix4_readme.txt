5.0 Platform Support

This fix is supported on all platforms supported by webMethods
Designer Process Development.



6.0 Cautions and Warnings

- You should restart Eclipse at least once with -clean parameter after fix installation
e.g.:

eclipse.exe -clean

- Do not use the Overinstall parameter supplied with Update
Manager. This can lead to problems with further updates within
Eclipse.



7.0 Fix Contents

WMDPD-10397
Business process is not initiated created in MWS admin page. Product issue.
The issue is resolved.
 
WMDPD-10440
Designer uses an older versionWMDPD-10397
Business process is not initiated created in MWS admin page. Product issue.
The issue is resolved. of the Batik library, which has known security vulnerabilities.
The issue is resolved. Designer now uses Batik 1.17.

WMDPD-10372
Vulnerable 3rd party component Xalan used
The issue is resolved.

WMDPD-10324
Vulnerable 3rd party component Apache Xmlgraphics batik used
The issue is resolved.


WMDPD-10346 
CLONE - Error when using data type as logged field
Custom field values will be read from logical server on rebuild of a process.

Issue is now resolved.

WMDPD-10212(DES_10.11_Process_Fix1)
Trigger error: Selector string parser error:Lexical error at line 1, column 3. Encountered: "%" (37), after : ""
Based on mutual agreement with UM team the UM filter string has been modified to remove '%' and replace '/' with a '.'.

The issue is resolved.

WMDPD-10262(DES_10.11_Process_Fix1)
PRT Error when Complex Join is used
In Designer, when step is configured to use complex join, the join expression may contain an extra brace.

The issue is resolved.

WMDPD-10302(DES_10.11_Process_Fix1)
Designer Process Development 10.11 Process Inputs tab appear empty on Debug Configuration
The Process Input tab was not disposed properly.

Now the issue is resolved.

8.0 Installation

1. Shut down Designer

2. Install the fix using the Software AG Update Manager.
For instructions, see Using the Software AG Update Manager
located either in the _documentation directory or on the
documentation Web site at http://documentation.softwareag.com.

Fix should appear in Update Manager as:

Designer > Process Development > Designer BPM Fix

3. Verify new BPM feature versions are installed by going to
Help > About Software AG Designer > Installation Details >
Features tab > Feature Id sort field

you should see for example:

webMethods Process Development 10.11.0.0001-....
com.webmethods.designer.process.feature



9.0 Uninstallation

Uninstall the fix using the Software AG Update Manager.
For instructions, see Using the Software AG Update Manager.

NOTE: This uninstall procedure can only be used to uninstall the
most recently installed fix. This action will revert your
installation to the previously installed fix if the prior fix was
installed using Software AG Update Manager, otherwise Eclipse
will be reverted to the BPM 10.11 GA version of the plugins
(10.11.0.0000-....)

You cannot apply this uninstall procedure to the previously
installed fix.


10.0 Globalization

This fix conforms to the internationalization standards of the
webMethods product suite and includes support for operation in
any country, locale, or language as specified in the webMethods
Installation Guide.It was not tested with non-English
configurations and non-ASCII data. However, this fix has no
globalization impact and can be applied to systems running in any
supported locale or configuration.


10.1 Localization

This fix does not require an updated Language Pack. It might
contain new messages and these messages will appear in English.
