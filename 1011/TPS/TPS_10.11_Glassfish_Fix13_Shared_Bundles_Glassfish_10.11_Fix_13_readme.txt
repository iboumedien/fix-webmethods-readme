1.0 Fix Name

webMethods Shared Libraries Parsers 10.11 Fix 11



2.0 Fix ID

TPP_10.11_Parsers_Fix11



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.11 webMethods products.



4.0 Requirements and Dependencies

Fixes are cumulative. Therefore, for each of the listed fixes, you can install the specified fix or 
later.

For language pack requirements, see the Globalization section of the Software AG Empower Product 
Support website at https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

webMethods Shared Libraries contains the outdated third party library Jackson Databind with version 2.15.3.

The issue is resolved. The following Jackson libraries are now updated:
The jackson-core library is updated from version 2.15.3 to version 2.16.1.
The jackson-annotations library is updated from version 2.15.3 to version 2.16.1.
The jackson-databind library is updated from version 2.15.3 to version 2.16.1.
The jackson-dataformat-yaml library is updated from 2.15.3 to version 2.16.1.
The jackson-dataformat-cbor library is updated from version 2.15.3 to version 2.16.1.
The jackson-module-jaxb-annotations library is updated from version 2.15.3 to version 2.16.1.
The jackson-module-jsonschema library is updated from version 2.15.3 to version 2.16.1
The jackson-datatype-jsr310 library is updated from version 2.15.3 to version 2.16.1.
The jackson-jaxrs-base library is updated from version 2.15.3 to version 2.16.1.
The jackson-jaxrs-json-provider library is updated from version 2.15.3 to version 2.16.1.

TPS-2039 (TPS_10.11_Glassfish_Fix8)

wMFix.MFTServer fix fails to install because of a missing package from jakarta-mail.

webMethods Shared Bundles contains the third party woodstox-core with version 6.4.0 which breaks warnings-as-errors builds due to missing Resolution.OPTIONAL enumeration.

The issue is resolved. The following library has been updated and the package has been added:
> jakarta-mail is updated from v1.6.7 to v1.6.7.1
> woodstox-core is updated from v6.4.0 to v6.5.0
