1.0 Fix Name

webMethods Shared Libraries Swagger 10.11 Fix 6



2.0 Fix ID

TPP_10.11_Swagger_Fix6



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.11 webMethods products.



4.0 Requirements and Dependencies

Fixes are cumulative. Therefore, for each of the listed fixes, you can install the specified fix or 
later.

For language pack requirements, see the Globalization section of the Software AG Empower Product 
Support website at https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-2554 (TPP_10.11_Swagger_Fix6)

webMethods Shared Libraries contains the outdated third party OpenAPI libraries which has known security vulnerabilities.

The issue is resolved.
The SnakeYAML library is now updated from version 2.0.0 to version 2.2.0.
The jackson-datatype-jsr310 library is now updated from version 2.15.2 to version 2.15.3.
The swagger-annotations.jar is now update from version 2.1.13 to 2.1.19
The swagger-core.jar is now updated from version 2.2.9 to 2.2.19
The swagger-models.jar is now updated from version 2.2.9 to 2.2.19
The swagger-parser-core.jar is now updated from version 2.1.13 to 2.1.19
The swagger-parser-v3.jar is now updated from version 2.1.13 to 2.1.19

TPS-2360 (TPP_10.11_Swagger_Fix5)

webMethods Shared Libraries contains the outdated third party library Jackson JSR310 with version 2.15.0-rc3.

The issue is resolved.
The jackson-datatype-jsr310 library is updated from version 2.15.0-rc3 to version 2.15.2.

TPS-2141 (TPP_10.11_Swagger_Fix4)

webMethods Shared Libraries contains an outdated versions of the jackson-datatype-jsr310 and OpenAPI libraries, as well as
SnakeYAML library version 1.33, which has known security vulnerabilities.

The issue is resolved.
The SnakeYAML library is now updated from version 1.33 to version 2.0.
The jackson-datatype-jsr310 library is now updated from version 2.14.0-rc2 to version 2.15.0-rc1.
The swagger-parser-v3 library is now updated from version 2.0.33 to version 2.1.13.
The swagger-parser-core library is now updated from version 2.0.33 to version 2.1.13.
The swagger-core library is now updated from version 2.2.0 to version 2.2.9.
The swagger-models library is now updated from version 2.2.0 to version 2.2.9.
The swagger-annotations library is now updated from version 2.2.0 to version 2.2.9.


TPS-1903 (TPP_10.11_Swagger_Fix3)

webMethods Shared Libraries contains the third party Snakeyaml with version 1.30, which has known security vulnerabilities.

Updated Snakeyaml to v.1.33

Updated the following Swagger jars:

> swagger-annotations.jar - updated from version 1.6.2 to version 1.6.6
> swagger-core.jar - updated from version 1.6.2 to version 1.6.6
> swagger-models.jar - updated from version 1.6.2 to version 1.6.6
> swagger-parser.jar - updated from version 1.0.54 to version 1.0.59

Updated the following Swagger OpenAPI jars:

> swagger-parser-v3.jar - updated from version 2.0.17 to version 2.0.33
> swagger-parser-core.jar - updated from version 2.0.17 to version 2.0.33
> swagger-core.jar - updated from version 2.1.1 to version 2.2.0
> swagger-models.jar - updated from version 2.1.1 to version 2.2.0
> swagger-annotations.jar - updated from version 2.1.1 to version 2.2.0
