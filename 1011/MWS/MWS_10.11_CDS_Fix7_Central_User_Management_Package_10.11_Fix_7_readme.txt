7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

MWS-30016 (MWS_10.11_CDS_Fix7)
The Common Directory Services administration interfaces in Integration Server do not provide an
option for configuring LDAP or database attributes for users and groups.

The issue is resolved.



MWS-32374 (MWS_10.11_CDS_Fix7)
The administration interfaces for Common Directory Services in Integration Server do not allow
saving searches for users, groups, and roles.

The issue is resolved.




MWS-32767 (MWS_10.11_CDS_Fix7)
When deleting LDAP attributes through the administration pages for Common Directory Services in
Integration Server the deletion progress dialog box becomes unresponsive without completing the
action.

The issue is resolved.




MWS-33016 (MWS_10.11_CDS_Fix7)
When updating user passwords on the he Common Directory Services administration pages inIntegration Server CDS might fail to validate and store password changes without logging errormessages.  The issue is resolved.

MWS-30018 (MWS_10.11_CDS_Fix6)
The Common Directory Services administration interfaces in Integration Server do not provide an
option for configuring LDAP or database attributes for users and groups.

The issue is resolved.




MWS-31628 (MWS_10.11_CDS_Fix6)
Common Directory Services logs an NoSuchMethodException error message in the server.log file for
Integration Server. This issue occurs when you install the WmCDS package on Integration Server
and create a diagnostic.zip file.

The issue is resolved.




MWS-31942 (MWS_10.11_CDS_Fix6)
A security issue with the third-party component moment.js was discovered in the WmCDS and
WmTaskEngine packages for Integration Server.

The issue is resolved.




MWS-32380 (MWS_10.11_CDS_Fix6)
The graphical user interfaces for Common Directory Services in Integration Server might display
placeholder texts instead of field and button labels. This issue occurs when accessing a
configuration page through a direct URL.

The issue is resolved.




MWS-32470 (MWS_10.11_CDS_Fix6)
The Common Directory Services configuration pages in Integration Server display an error message
when you attempt to view the details of a group. This issue occurs when the LDAP directory
service for the group is configured with the Do Not Cache option.

The issue is resolved.




MWS-32653 (MWS_10.11_CDS_Fix6)
Modal dialogues on the administration pages for Common Directory Services in Integration Server do
not get closed when the user session expires.

The issue is resolved.




MWS-31728 (MWS_10.11_CDS_Fix5)

option for configuring a custom password complexity class.

8.0 Installation

Install using Software AG Update Manager. For instructions, see the documentation at 
http://documentation.softwareag.com.

If this is the first time you are installing a fix using the Software AG Update Manager, stop 
My webMethods Server and back up the My webMethods Server database and file system before 
applying the fix.


9.0 Uninstallation

These instructions can only be used to uninstall the most recently installed fix. This action 
will revert your installation to the previously installed fix. You cannot use these instructions to 
uninstall the previously installed fix.

Uninstall using Software AG Update Manager. For instructions, see the documentation at 
http://documentation.softwareag.com.

You can only uninstall one My webMethods Server fix at a time. Make sure you backup your server 
settings before uninstalling My webMethods Server fixes.																								

10.0 Globalization

This fix conforms to the internationalization standards of the webMethods product suite and includes 
support for operation in any country, locale, or language as specified in the Installing webMethods 
Products guide. It was not tested with non-English configurations and non-ASCII data. However, this 
fix has no globalization impact and can be applied to systems running in any supported locale or 
configuration.


10.1 Localization

This fix does not require an updated language pack. It might contain new messages and these messages 
will appear in English.
