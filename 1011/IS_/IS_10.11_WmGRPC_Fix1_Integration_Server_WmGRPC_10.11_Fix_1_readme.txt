webMethods Integration Server 10.11 WmGRPC Fix 1.



2.0 Fix ID

IS_10.11_WmGRPC_Fix1



3.0 Product(s)/Component(s) Affected

This fix affects webMethods Integration Server 10.11.



4.0 Requirements and Dependencies

For language pack requirements, see the Globalization section of the Software AG Empower Product 
Support website at https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

webMethods Integration Server 10.11 WmGRPC Fix 1 is supported on the 
same platforms and operating systems and with the same JREs as webMethods 
Integration Server 10.11.



6.0 Cautions and Warnings

If Integration Server uses central user management which is provided through 
My webMethods Server, you must keep the common libraries provided by My webMethods 
Server up to date. To obtain the current changes to the common libraries, install
the latest My webMethods Server fix located under Common Libraries in Software AG 
Update Manager.



7.0 Fix Contents

PIE-72876
Upgrade of gRPC libraries to remediate security vulnerabilities, including
vulnerabilities in the Netty libraries. 

The gRPC libraries installed with the optional WmGRPC package have been upgraded
to remediate the following security vulnerabilities: 

CVE-2021-37137 https://nvd.nist.gov/vuln/detail/CVE-2021-37137
CVE-2021-37136 https://nvd.nist.gov/vuln/detail/CVE-2021-37136
CVE-2020-11612 https://nvd.nist.gov/vuln/detail/CVE-2020-11612
CVE-2019-20445 https://nvd.nist.gov/vuln/detail/CVE-2019-20445
CVE-2019-20444 https://nvd.nist.gov/vuln/detail/CVE-2019-20444

The old libraries are:

grpc-api-1.27.0
grpc-context-1.27.0
grpc-core-1.27.0
grpc-netty-1.27.0
grpc-netty-shaded-1.27.0
grpc-protobuf-1.27.0
grpc-protobuf-lite-1.27.0
grpc-stub-1.27.0

The new libraries are:

grpc-api-1.45.1
grpc-context-1.45.1
grpc-core-1.45.1
grpc-netty-1.45.1
grpc-netty-shaded-1.45.1
grpc-protobuf-1.45.1
grpc-protobuf-lite-1.45.1
grpc-stub-1.45.1

After the library upgrades, gRPC processing in Integration Server uses Netty
version 4.1.72.Final. 

In addition to including resolutions to product defects, a fix may also include
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters. 
Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the
release.

10.0 Globalization

This fix conforms to the internationalization standards of the webMethods product
suite and includes support for operation in any country, locale, or language as
specified in the Installing webMethods Products guide. It was not tested with 
non-English configurations and non-ASCII data. However, this fix has no
