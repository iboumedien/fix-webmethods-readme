6.0 Cautions and Warnings

All product profiles that run on top of OSGi runtimes are stopped by
the Software AG Update Manager and will automatically be started
after the session completes.



7.0 Fix Contents

This fix is cumulative, which means that it is an aggregate of previous fixes 
for the release.

LAR-1196
Common Landscape Asset Registry uses the third-party library
Eclipse JGit 6.9.0, which has security vulnerabilities.
The issue is resolved. Common Landscape Asset Registry now uses Eclipse JGit 7.2.1.

LAR-1159
Common Landscape Asset Registry uses the third-party library
Eclipse JGit 6.7.0, which has security vulnerabilities.
The issue is resolved. Common Landscape Asset Registry now uses Eclipse JGit 6.9.0.

LAR-1160
Common Landscape Asset Registry uses the third-party library
Google Code Gson 2.9.0, which has security vulnerabilities.
The issue is resolved. Common Landscape Asset Registry now uses Google Code Gson 2.10.1.
