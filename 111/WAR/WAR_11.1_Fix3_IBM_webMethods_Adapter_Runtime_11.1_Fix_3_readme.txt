4.0 Requirements and Dependencies

This fix requires the installation of the latest Integration Server Core fix.

Fixes are cumulative. Therefore, for each of the listed fixes, you can install 
the specified fix or later.



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include 
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters. 
Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the 
release.

PIEAR-2963
Adoption of Jakarta namespace changes for XAResource.

PIEAR-2779 (WAR_11.1_Fix2)
As part of the webMethods rebranding, all the Software AG references are updated to webMethods.

PIEAR-2784 (WAR_11.1_Fix2)
Adoption of Jakarta namespace changes.

PIEAR-2684 (WAR_11.1_Fix1)
The Adapter health check API has been enhanced to remove the Spring Boot dependency.



8.0 Installation

Use IBM webMethods Command Central to install fixes on multiple machines from a central location (9.7 or later), 
or IBM webMethods Update Manager to install fixes on one machine at a time. For instructions,
see the documentation at https://www.ibm.com/docs/en/products.



9.0 Uninstallation

Note: These instructions can only be used to uninstall the most recently installed fix. This action 
will revert your installation to the previously installed fix. You cannot use these instructions to 
uninstall the previously installed fix.

Uninstall using IBM webMethods Command Central (9.7 or later) or IBM webMethods Update Manager. For instructions, see 
the documentation at https://www.ibm.com/docs/en/products.
