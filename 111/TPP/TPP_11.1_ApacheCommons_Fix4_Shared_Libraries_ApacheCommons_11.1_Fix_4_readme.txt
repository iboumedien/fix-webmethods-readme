1.0 Fix Name

webMethods Shared Libraries ApacheCommons 11.1 Fix 4



2.0 Fix ID

TPP_11.1_ApacheCommons_Fix4



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 11.1 webMethods products.



4.0 Requirements and Dependencies

None.



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-4777 (TPP_11.1_ApacheCommons_Fix4)

webMethods Shared Libraries contains an outdated version of the third party Apache Commons Lang3 library,
which has known security vulnerabilities.

The issue is resolved. The following library has been updated:
The commons-lang3 library is updated from version 3.17.0 to version 3.18.0.

TPS-4729 (TPP_11.1_ApacheCommons_Fix3)

webMethods Shared Libraries contains an outdated version of the third party Apache Commons FileUpload library, 
which has known security vulnerabilities.

The issue is resolved. The following libraries have been updated:
The commons-fileupload library is updated from version 1.5.0 to version 1.6.0.
The commons-io library is updated from version 2.15.1 to version 2.19.0.

TPS-4713 (TPP_11.1_ApacheCommons_Fix2)

webMethods Shared Libraries contains an outdated version of the third party Apache Commons Lang3 library.

The issue is resolved. The following library has been updated:
The commons-lang3 library is updated from version 3.14.0 to version 3.17.0.

TPS-4697 (TPP_11.1_ApacheCommons_Fix1)

webMethods Shared Libraries contains an outdated version of the third party Apache Commons BeanUtils library, 
which has known security vulnerabilities.

The issue is resolved. The following library has been updated:
The commons-beanutils library is updated from version 1.9.4 to version 1.11.0.
