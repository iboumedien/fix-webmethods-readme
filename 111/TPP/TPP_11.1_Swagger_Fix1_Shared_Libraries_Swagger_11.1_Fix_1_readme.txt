TPS-4801 (TPP_11.1_Swagger_Fix1)

webMethods Shared Libraries contains outdated versions of the third party OpenAPI Swagger libraries,
which have known security vulnerabilities.

The issue is resolved. The following libraries have been updated:
The swagger-annotations library is updated from 2.2.19 to version 2.2.36.
The swagger-core library is updated from 2.2.19 to version 2.2.36.
The swagger-models library is updated from 2.2.19 to version 2.2.36.
The swagger-parser-core library is updated from 2.1.19 to version 2.1.32.
The swagger-parser-v3 library is updated from 2.1.19 to version 2.1.32.
The snakeyaml library is updated from version 2.2.0 to version 2.4.0.

Additionally, the following library has been newly included:
The swagger-parser-safe-url-resolver library has been added at version 2.1.32.
