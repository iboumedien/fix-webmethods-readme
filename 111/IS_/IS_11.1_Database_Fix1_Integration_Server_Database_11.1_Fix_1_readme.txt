7.0 Fix Contents

PIE-95631
Database migration from 10.15 to 11.1 fails.
 
An error similar to the following occurs when migrating databases from 10.15 to
11.1: 

error: path not found for 10.15.fix5 SQL_SERVER to latest

This issue is now resolved.


The fix will change the structure of the database. Review the fix contents carefully before installing
the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.
