4.0 Fix Contents

ZSL-313 (ZSL_11.1_Fix2)

ZSL contains the third party apache tika core v2.8.0 which has been reported as vulnerable.

The issue is resolved. Following libraries have been updated to a newer version:
apache-tika-core is updated from v2.8.0 to v3.2.3


5.0 Platform Support

the same platforms and operating systems and with the same JREs
as Central Security Library 11.1.
