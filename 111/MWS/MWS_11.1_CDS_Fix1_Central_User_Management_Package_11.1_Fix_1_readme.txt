7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

MWS-37333 (MWS_11.1_CDS_Fix1)
MWS-37373 (MWS_11.1_CDS_Fix1)
A potential HTTP Response Splitting isssue in WmCDS package has been fixed.


MWS-37493 (MWS_11.1_CDS_Fix1)
Under certain circumstances when working with users/roles/groups CDS UI in the IS might display
the following errors: "cannot store to object array because "this.m_pageControls" is null" or
"cannot store to object array because "this.m_pageControls" is null".


MWS-37666 (MWS_11.1_CDS_Fix1)
CCE shows a wrong status for WmCDS component. CCE shows a wrong status for WmCDS component and thefollowing error is observed at logs: "No subsystem with id WmCDS is found for runtime componentwith id OSGI-IS_5000-WmCDS" The issue is fixed.
