7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

WPE-5387
Ð¢he IDX_PRTPROCESS_ROOT and IDX_PRTPROCESS_PARENT indexes are now created in the PRTPROCESS table of the Process Engine database.

WPE-5300
Database Component Configurator fails to migrate the Process Engine component from 10.5 to 10.15. The issue occurs when the component is a MySQL database.

The issue is resolved.

WPE-5441
In webMethods Process Engine, querying the Process Audit Log database component may cause performance issues.

The issue is now resolved. This fix creates a new index in the DEPLOYMENTVERSION column of the WMSTEPDEFINITION table in the Process Audit schema.

WPE-5452
In webMethods Process Engine, performing a search from the Process Instances page may cause performance issues.

The issue is resolved. This fix creates two new indices in the PRA_PROCESS_RECENT table in the Process Audit schema.

WPE-5474
With webMethods Process Engine, a high database CPU utilization situation may occur.
When Process Engine queries PRTPROCESS table, high utilization of database CPU is observed.
The issue is now resolved. This fix creates a new index in the PARENTSTEPTYPE and ROOTINSTANCEID
columns of the PRTPROCESS table in the Process Engine schema.
