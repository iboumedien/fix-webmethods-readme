1.0 Fix Name

WmDesigner 11.1 Fix 1 for Deployer



2.0 Fix ID

DES_Deployer_11.1_Fix1



3.0 Product(s)/Component(s) Affected

This fix affects IBM webMethods Designer for Deployer (WmDesigner package).



4.0 Requirements and Dependencies

None.



5.0 Platform Support

This fix is supported on the same platforms as IBM webMethods Designer 11.1, 
IBM webMethods BPM Process Development 11.1, and IBM webMethods Deployer 11.1.



6.0 Cautions and Warnings

You should install this fix on all servers that host the WmDesigner package, 
including the IBM webMethods Deployer source and target servers.



7.0 Fix Contents

WMDPD-10790
There are CSS styling problems with the Logical Servers tab 
in the WmDesigner package page.
 
The issue is resolved.


8.0 Installation

Use Command Central to install fixes on multiple machines from a central location 
(9.7 or later products), or Update Manager to install fixes on one machine at a time. 
For instructions, see the documentation at https://docs.webmethods.io/.

In Update Manager, this fix is located under Integration Server > WmDesigner Fix.


9.0 Uninstallation

Note: These instructions can only be used to uninstall the most recently installed fix. This action 
will revert your installation to the previously installed fix. You cannot use these instructions to 
uninstall the previously installed fix.

Uninstall using Command Central (9.7 or later) or Update Manager. For instructions, see 
the documentation at https://docs.webmethods.io/.
