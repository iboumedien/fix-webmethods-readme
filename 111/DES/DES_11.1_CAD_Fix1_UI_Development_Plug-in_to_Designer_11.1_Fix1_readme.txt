7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

MWS-37324 (DES_11.1_CAD_Fix1)
The built-in encryption method in My webMethods Server is not sufficiently secure.

The issue is resolved. My webMethods Server now uses SHA265 hashing for encryption.


MWS-37347 (DES_11.1_CAD_Fix1)
The product uses insufficiently random numbers or values in a security context that depends on
unpredictable numbers. The issue is resolved.


MWS-37369 (DES_11.1_CAD_Fix1)
We are now storing logs in log files instead of in the console output.


WMC-5359 (DES_11.1_CAD_Fix1)
The previous Task Type ID was based on the old algorhytm, resulting in an error. The issue isresolved.
