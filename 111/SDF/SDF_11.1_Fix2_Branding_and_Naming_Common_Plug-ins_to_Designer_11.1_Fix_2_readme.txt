7.0 Fix Contents


SDEF-4333
Upgraded SnakeYAML and Jackson libraries to address security vulnerabilities. 

Designer now uses the upgraded versions of the following libraries to address 
the known security vulnerabilities:

SnakeYAML: upgraded from 2.2.0 to 2.4.0.
Jackson: upgraded from 2.16.1 to 2.19.2.


SDEF-4228/SDEF-4205 (SDF_11.1_Fix1)
Resolved multiple security vulnerabilities in the Designer platform.

This fix addresses the following categories of vulnerabilities within the 
Designer platform:

•	Do not compare class objects with getName() or getSimpleName() methods
•	Information exposure through error messages
•	Insecure SSL
•	Path Relative Stylesheet Import
•	Plaintext storage of Password
•	XML entity Injection rule
