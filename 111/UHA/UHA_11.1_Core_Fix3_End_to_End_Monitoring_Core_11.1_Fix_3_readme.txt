6.0 Cautions and Warnings

Applying a fix reverts the changes in these files to their default values. "E2EMonitoring\agent\config\agent.config",
"E2EMonitoring\agent\config\apm-trace-include-plugin.config" and "E2EMonitoring\agent\e2ecustomlog4j2.properties".
Create a backup of these files before applying the fix, if you have made changes to these files. After the fix is 
applied, reapply the changes.
