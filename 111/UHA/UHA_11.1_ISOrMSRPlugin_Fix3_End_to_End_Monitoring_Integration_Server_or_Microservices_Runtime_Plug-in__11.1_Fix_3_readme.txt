8.0 Installation

Install using IBM webMethods Update Manager. For instructions, see the documentation at 
https://docs.webmethods.io/.

1. If this is the first time you are installing a fix using the IBM webMethods Update Manager, 
   back up the IBM webMethods End-to-End Monitoring file system before applying the fix.   
   
2. After fix installation, copy \E2EMonitoring\agent\plugins\uha-onpremise-is-plugin.jar to
   \IntegrationServer\lib\jars directory.



9.0 Uninstallation

Note: These instructions can only be used to uninstall the most recently installed fix. This action will 
revert your installation to the previously installed fix. You cannot apply these instructions to the 
previously installed fix.

1. Uninstall using the Software AG Update Manager. For instructions, see the documentation 
   at http://documentation.softwareag.com.

2. After fix uninstallation, To revert the IBM webMethods product installation file system to the state before fix installation,
   delete changes made to the IBM webMethods product installation file system, along with this jar file \IntegrationServer\lib\jars\uha-onpremise-is-plugin.jar
