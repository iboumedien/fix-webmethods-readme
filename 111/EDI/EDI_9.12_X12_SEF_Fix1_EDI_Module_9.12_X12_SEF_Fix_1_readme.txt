7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters. 
Review the fix contents carefully before applying the fix.

This fix is cumulative, which means that it is an aggregate of previous fixes 
for the release.

EEO-4645

The sef file was not containing description and an alternate name of a segment.
This issue is now resolved and the sef file now contains a description and an
alternate name of a segment. Currently, sef schema with description and an
alternate name of a segment is supported for only standard X12 version 7050,
7060, and 8010.
To use sef file with description, please set EDI Configuration property
InstallSEFSchemaWithDescription as true along with EDI Configuration property 
EDISchemaSource as SEF. When InstallSEFSchemaWithDescription set as true while 
Creating a Flat File Schema from a SEF File the sef file is read from this
location packages\WmEDI\pub\SEFS\X12_SefDesc
instead of packages\WmEDI\pub\SEFS\X12.

10.0 Globalization

This fix conforms to the internationalization standards of the webMethods 
product suite and includes support for operation in any country, locale, or 
language as specified in the Installing webMethods Products guide. It was not 
tested with non-English configurations and non-ASCII data. However, this fix
