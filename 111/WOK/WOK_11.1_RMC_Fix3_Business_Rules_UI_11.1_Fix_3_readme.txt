7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in 
services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

WOK-19540 (WOK_11.1_RMC_Fix1)
Slow MWS UI pagination with large decision tables
When working with large decision tables, the pagination response times of the Business Rules UI on 
My webMethods Server can be very slow.
This issue has been resolved by improving the UI response times when paging within a decision 
table..

WOK-20101 (WOK_11.1_RMC_Fix2)
CVE-2020-7760 for codemirror-4.7.0
CVE-2020-7760 was identified for used library codemirror-4.7.0.
This issue is resolved by upgrading to codemirror-6.0.1

WOK-20228 (WOK_11.1_RMC_Fix2)
Duplicate "Business Rules" and "webMethods Business Rules" navigation entries
After upgrading to 11.1, the RMC nav tree will show duplicate navigation entries under "Business 
Rules", "webMethods Business Rules", "webMethods Business Rules Settings" and "Business Rules 
Settings".
This issue has been resolved by removing the obsolete navigation entries.

WOK-20354
ClassCastException when an element of a document list used as DT input is null
If a decision table uses a document list as one of its inputs and one element of this document list 
is 'null' at runtime, a ClassCastException may occur when invoking the decision table in the 
Business Rules Runtime.
This issue has been resolved.
