5.0 Fix Contents

MWS-26562

Building CAF applications with Asset Build Environment fails. 
This issue occurs when the application project contains different
versions of third party libraries that are also included in the 
My webMethods Server distribution.

The issue is resolved.

PIE-58258

When a My webMethods Server (MWS) user defined in common directory 
services is mapped to a certificate on Integration Server, running
the Asset Build Environment (ABE) results in the error "User does
not exist for certificate”.

This issue is now fixed.
