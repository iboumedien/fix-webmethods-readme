7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include 
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters. 
Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the 
release.

JDS-61
Updated the Product name as per IBM rebranding and also Licenses 3rdparty 
notices for fix JDS_10.2_Fix3.


JDS-30 (JDS_10.2_Fix2)
Upgraded Apache Spark Progress driver to 6.0.1.0124 version for compatibility
with Java 17.


JDS-4 (JDS_10.2_Fix1)
Upgraded the Progress driver to version 6.0.4 due to a vulnerability in HttpClient
versions prior to 4.5.13 and 5.0.3. This vulnerability causes a misinterpretation
of the malformed authority component in request URIs passed to the library as
java.net.URI object and pick the wrong target host for request execution.

This issue is now resolved.



8.0 Installation
