1.0 Fix Name

Terracotta DB 10.5.0 Fix 5


2.0 Fix ID

TDB_10.5.0_Fix5


3.0 Product(s)/Component(s) Affected

This fix affects Terracotta DB 10.5.0 components


4.0 Requirements

None.


5.0 Platform Support

This fix is supported on all the platforms that Terracotta DB 10.5.0 supports.


6.0 Cautions and Warnings

7.0 Fix Contents


    [TDB-5591] - Vulnerable 3rd Party Component spring used, update to 5.3.18


8.0 Installation

1. Shut down all Terracotta DB servers. A safe shutdown procedure is as follows.

a. Shut down the mirror servers.

b. Shut down the clients. A Terracotta client will shut down when you shut down
   your application.
   
c. Shut down the active servers.

   
3. Install the fix using the Software AG Update Manager.
For instructions, see Using the Software AG Update Manager
located either in the _documentation directory or on the 
documentation Web site at http://documentation.softwareag.com.


9.0 Uninstallation

1. Shut down all Terracotta DB servers. A safe shutdown procedure is as follows.

a. Shut down the mirror servers.

b. Shut down the clients. A Terracotta client will shut down when you shut down
   your application.
   
c. Shut down the active servers.

2. Uninstall the fix using the Software AG Update Manager.
For instructions, see Using the Software AG Update Manager.

NOTE: This uninstall procedure can only be used to uninstall the
most recently installed fix. This action will revert your
installation to the previously installed fix. You cannot apply
this uninstall procedure to the previously installed fix.
