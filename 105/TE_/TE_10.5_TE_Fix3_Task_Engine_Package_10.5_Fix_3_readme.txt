7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

WTN-5216(TE_10.5_TE_Fix3)
Task Engine on Integration Server might not parse task schedule dates correctly and fail to queue 
task instances form a type. This issue occurs when selecting the schedule date from the calendar 
picker in the Task Engine UI.

The issue is resolved. 


WTN-5219 (TE_10.5_TE_Fix3)
The SPM plug-in for Task Engine on Integration Server might display NullPointerException error messages. 
This issue occurs when Integration Server is not running, and the Common Directory Services component 
is not configured.

The issue is resolved. 

WTN-5331 (TE_10.5_TE_Fix3)
The wmTaskEngine package in Integration Server might attempt to initialize some of the task services 
and log initialization errors, even when disabled through the custom_wrapper.conf file for 
Integration Server.

The issue is resolved. Now setting the -Dtask.library.mode.enable property to false in the 
custom_wrapper.conf file of Integration Server completely disables the initialization of the 
wmTaskEngine package. 

WTN-5060(TE_10.5_TE_Fix2)
Task Engine might pick up incorrect localization settings when used with non-English locale, 
and display Active Transfer UI strings. This issue occurs when Task Engine is installed on 
Integration Server.

The issue is resolved. 

WTN-5117(TE_10.5_TE_Fix2)
User task steps in a process never transition to status Completed when Task Enigne is running
 on Integration Server.

The issue is resolved. 

WTN-5216(TE_10.5_TE_Fix2)
Task Engine on Integration Server might not parse task schedule dates correctly and fail to 
queue task instances form a type. This issue occurs when selecting the schedule date from 
the calendar picker in the Task Engine UI.

The issue is resolved. 


MWSSOF-2399 (TE_10.5_TE_Fix1)
The graphical user interface for Task Engine instances, running on Integration Server does not 
provide options to administer and monitor task types.

The issue is resolved. The wmTaskEngine package now provides a graphical user interface for task 
type administration.
 
MWSSOF-2427 (TE_10.5_TE_Fix1)
Task Engine instances, running on Integration Server do not provide options to administer and 
monitor task instances.

The issue is resolved. The wmTaskEngine package now provides a graphical user interface for task 
instance management.

MWSSOF-2473 (TE_10.5_TE_Fix1)
Task Engine instances, running on Integration Server provide limited options to configure logging.

The issue is resolved. The wmTaskEngine package now provides an enhanced logging configuration
interface.


8.0 Installation

Install using Software AG Update Manager. For instructions, see the documentation at

9.0 Uninstallation

Note: These instructions can only be used to uninstall the most recently installed fix. This action 
will revert your installation to the previously installed fix. You cannot use these instructions to 
uninstall the previously installed fix.

You can only uninstall one Task Engine fix at a time. Make sure you backup your server settings 
before uninstalling Task Engine fixes.



10.0 Globalization

This fix conforms to the internationalization standards of the webMethods product suite and includes 
support for operation in any country, locale, or language as specified in the Installing webMethods 
Products guide. It was not tested with non-English configurations and non-ASCII data. However, this 
fix has no globalization impact and can be applied to systems running in any supported locale or 
configuration.
