webMethods Shared Libraries Misc 10.5 Fix 3



2.0 Fix ID

3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.5 webMethods products.

6.0 Cautions and Warnings

7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-1310 (TPP_10.5_Misc_Fix3)

WebMethods Shared Libraries contains an old version of Joda Time library (v1.6.2) for which is reported the following issue:
"[PIE-71947] When creating Complex Scheduled Task on IS Scheduler , the Tasks are running on wrong time;
This is applies for the newly created tasks , and for the tasks that already exist."

The issue is resolved. Joda Time library is updated from v1.6.2 to v2.10.12 and the reported issue doesn't appear anymore.


TPS-738 (TPP_10.5_Misc_Fix2)

Some WebMethods products depend on Maverick SFTP client library v1.7.23, which currently is not available 
at the common shared location.

The issue is resolved. The Maverick SFTP client library with v1.7.23 is now available at following shared
location: common/lib/ext and could be used by the WebMethods products.


TPS-687 (TPP_10.5_Misc_Fix1)

WebMethods Shared Libraries contains an old version of JSch - Java Secure Channel (v0.1.54) for which
is reported the following error:
com.jcraft.jsch.JSchException: Session.connect: java.security.SignatureException: Invalid encoding for signature" 

The issue is resolved. The jsch library is updated from v0.1.54 to v0.1.55.
