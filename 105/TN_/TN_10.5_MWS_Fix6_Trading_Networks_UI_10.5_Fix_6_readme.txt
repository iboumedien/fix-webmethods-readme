7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters. 
Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes 
for the release.

OTN-23017

MyWebMethods, some incoming xml ooders contain int CDATA section are freezing 
the MyWebMethods instance
This issue is now resloved

OTN-22642 (TN_10.5_MWS_Fix5)

Created a public search, where use this filter option and then selected it as 
a default search (options tab >Basic as a default search tab).
Open the Transaction page it is opened on the Advanced tab and not basic.
Switching to Basic clean the content and return to "All".
This issue is now resolved.

OTN-22492 (TN_10.5_MWS_Fix5)
In MyWebmethods Server, Sometimes a query triggers before the page displays when
opening the activity Log in MyWebmethods Server. This causes issues in the
database, as the query runs long and uses up all available temp tablespace.
This issue is now resolved.

OTN-21816 (TN_10.5_MWS_Fix4)
Used Java 8 API to handle DST issue.

OTN-21884 (TN_10.5_MWS_Fix4)
When a user logs into MWS with OAuth user, functionalities such as Submit
Documents, Configure Properties, Partner Group deletion, and Processing Rule
creation does not function properly. The user gets an HTTP error response on the screen.
This issue is resolved now.

OTN-21890 (TN_10.5_MWS_Fix4)
when you create any public queue and copy it using copy action and save it after
that whenever you try to create any new public queue it will always append
"Copy of" String in the newly created queue.

This issue is now resolved.

OTN-21532 (TN_10.5_MWS_Fix3)
When the magnifying glass icon of last column is clicked in my webMethods
Server, the message in Brief Message column of ActivityLogs page displays the
Swedish characters incorrectly.
This issue is now resolved.

OTN-21604 (TN_10.5_MWS_Fix3)
null

OTN-20662 (TN_10.5_MWS_Fix2)
Existing values of fields in Private Queue Details screen gets deleted.

The following sequence of actions results in removal of existing values of
fields in the Private Queue Details screen:
- Open a custom Delivery Method in edit mode.
- Click OK or Cancel to exit the edit mode.
- Open a 'Queue for delivery' Delivery Method.
- Click OK or Cancel to exit the edit mode.
- Click Save.

This issue is now resolved.

OTN-20732 (TN_10.5_MWS_Fix2)
An issue related to password masking has now been fixed in My webMethods Server.

OTN-20764 (TN_10.5_MWS_Fix2)
Activity log displays incorrect Chinese characters.

In My webMethods Server, the message in Brief Message column of
ActivityLogs page displays the Chinese characters incorrectly.

This issue is now resolved.

OTN-20512 (TN_10.5_MWS_Fix1)
In webMethods Trading Networks, for attributes of type String List, the 
Attributes section under Transactions is listing only a maximum of 10 elements 
even if there are more than 10 elements retrieved during document recognition.
This issue is resolved.

OTN-20594 (TN_10.5_MWS_Fix1)
Existing values of fields in Private Queue Details screen gets deleted.

The following sequence of actions results in removal of existing values of
fields in Private Queue Details screen:
- Open a custom Delivery Method in edit mode.
- Click OK or Cancel to exit the edit mode.
- Click Save.

This issue is now resolved.
