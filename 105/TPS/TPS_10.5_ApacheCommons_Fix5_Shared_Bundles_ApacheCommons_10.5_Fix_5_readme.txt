7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-1908 (TPS_10.5_ApacheCommons_Fix5)

The fix install instructions have been changed.

TPS-1275 (TPS_10.5_ApacheCommons_Fix4)

WebMethods Shared Bundles contains an old version of Apache Commons Fileupload (v1.3.3). Due to an unsatisfied dependency from com.webmethods.caf.server to java.package; org.apache.commons.fileupload [1.4.0,2.0.0), the Apache Commons Fileupload library need to be updated with a version between v1.4.0 and v2.0.0.

The issue is resolved. The Apache Commons Fileupload library is updated from v1.3.3 to v1.4.0.

TPS-1224 (TPS_10.5_ApacheCommons_Fix3)

WebMethods Shared Bundles contains an old version of Apache Commons IO (v2.6.0) which has
known security vulnerabilities.

The issue is resolved. The Apache Commons IO library is updated from v2.6.0 to v2.10.0.

TPS-1144 (TPS_10.5_ApacheCommons_Fix2)

WebMethods Shared Bundles contains an old version of Apache Commons Httpclient(v4.5.5) which has
known security vulnerabilities.

The issue is resolved. The Apache Commons Httpclient library is updated from v4.5.5 to v4.5.13.
The Apache Commons Httpcore library has also been updated from v.4.4.9 to v4.4.14.

TPS-756 (TPS_10.5_ApacheCommons_Fix1)

WebMethods Shared Bundles contains an old version of Apache Commons Beanutils (v1.9.3) which has
known security vulnerabilities.

The issue is resolved. The Apache Commons Beanutils library is updated from v1.9.3 to v1.9.4.
