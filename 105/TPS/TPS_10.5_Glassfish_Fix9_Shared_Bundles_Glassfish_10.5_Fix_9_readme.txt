1.0 Fix Name

webMethods Shared Bundles Glassfish 10.5 Fix 9



2.0 Fix ID

TPS_10.5_Glassfish_Fix9



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.5 webMethods products.



4.0 Requirements and Dependencies

Fixes are cumulative. Therefore, for each of the listed fixes, you can install the specified fix or 
later.

For language pack requirements, see the Globalization section of the Software AG Empower Product 
Support website at https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-1596 (TPS_10.5_Glassfish_Fix9)

webMethods Shared Bundles contains the third party Jackson Databind with version v2.12.3, which has known security vulnerabilities.

The issue is resolved. The following Jackson libraries have been updated:
> jackson-core is updated from v2.12.3 to v2.13.2
> jackson-annotations is updated from v2.12.3 to v2.13.2
> jackson-databind is updated from v2.12.3 to v2.13.2.2
> jackson-dataformat-yaml is updated from v2.12.3 to v2.13.2
> jackson-dataformat-cbor is updated from v2.12.3 to v2.13.2
> jackson-module-jaxb-annotations from v2.12.3 to v2.13.2
> jackson-jaxrs-base from v2.12.3 to v2.13.2
> jackson-jaxrs-json-provider from v2.12.3 to v2.13.2
> snakeyaml from v1.27.0 to v1.30.0

TPS-1270 (TPS_10.5_GlassFish_Fix7)
Due to the following issue:
"Missing requirement: cel spm-cel-impl module 10.5.0.0004-0194 (com.softwareag.plm.spm.cel.impl 10.5.0.0004-0194) requires 'java.package; com.fasterxml.jackson.core [2.12.0,3.0.0)' but it could not be found"
TPS.Glassfish fix instructions are updated to fix the reported missing requirement.

TPS-1256 (TPS_10.5_GlassFish_Fix6)
webMethods Shared Bundles contains the third party jersey-common with version 2.28.0, which have known security vulnerabilities.

The issue is resolved. Following Jersey libraries have been updated:
jersey-client.jar updated from 2.28.0 to 2.34.0
jersey-common.jar updated from 2.28.0 to 2.34.0
jersey-container-servlet-core.jar updated from 2.28.0 to 2.34.0
jersey-container-servlet.jar updated from 2.28.0 to 2.34.0
jersey-hk2.jar updated from 2.28.0 to 2.34.0
jersey-server.jar updated from 2.28.0 to 2.34.0
jersey-media-json-jackson.jar updated from 2.28.0 to 2.34.0
jersey-media-jaxb.jar updated from 2.28.0 to 2.34.0
jersey-entity-filtering.jar updated from 2.28.0 to 2.34.0

TPS-1196 (TPS_10.5_GlassFish_Fix5)

The issue is resolved. Following libraries have been updated:
