1.0 Fix Name

Software AG Common Platform 10.5 Fix 31



2.0 Fix ID

OGI_10.5_Fix31


3.0 Product(s)/Component(s) Affected


This fix affects OSGi Common Platform 10.5.



4.0 Requirements

This fix requires Software AG Common Platform 10.5. 
Software AG Common Platform 10.5 Fix 31 
must be installed together with TPS.Glassfish_10.5.0_Fix6 
if the products for these fixes are installed on the system. 

Fixes are cumulative. Therefore, for each of the listed fixes, you can install
the specified fix or later.


5.0 Platform Support

Same platforms as product release.


6.0 Cautions and Warnings

All product profiles that run on top of OSGi runtimes are stopped by
the IBM webMethods Update Manager and will automatically be started
after the session completes.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes
for the release.

OGI-4930
Common Platform uses the third-party Apache MINA Core 2.0.23, which has
known security vulnerabilities.
The issue is resolved. Common Platform now uses the
stable version Apache MINA Core 2.2.4

OGI-3914
Software AG Common Platform uses Apache Tomcat v. 8.5.95, which has security vulnerabilities.

documentation at http://documentation.softwareag.com.
