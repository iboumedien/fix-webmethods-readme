OSGi Common Platform Diagnostic Tool 10.5 Fix 15



2.0 Fix ID

3.0 Product(s)/Component(s) Affected


This fix affects OSGi Common Platform 10.5.



4.0 Requirements

This fix requires any of the products listed in 2.0 Products (s)/Component(s) Affected. 

Official fixes are cumulative. Therefore, you can install the specified fix or later.

For language pack requirements, see the Globalization section of the Software AG
Empower Product Support website at
https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

This fix is supported on all platforms supported
by webMethods Product Suite 10.5.



6.0 Cautions and Warnings

Software AG Tomcat Server is stopped by Software AG Update 
Manager, therefore it must be restarted after applying the fix.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters. 
Review the fix contents carefully before applying the fix.

This fix is cumulative, which means that it is an aggregate of previous fixes 
for the release.

OGI-3399
The Software AG Common Platform diagnostic tool uses a version of
Undertow that has known vulnerabilities.
This issue is resolved. Undertow dependency is removed.

OGI-3246
The Software AG Diagnostic Tool uses a version of the third-party jQuery library,
that has known vulnerabilities.
This issue is resolved. The Diagnostic Tool now uses jQuery version 3.6.0.

OGI-3148
The default configuration of the Software AG Common Platform diagnostic tool does
not include the fix-level profile and Software AG Update Manager installation logs if
Update Manager is not installed in the Software AG installation directory.
This issue is resolved. The default configuration now includes the fix-level profile and
you can specify the Software AG Update Manager installation directory when you run
the diagnostic tool. For details, see the Software AG Infrastructure Administrator’s Guide.

OGI-3175
The Software AG Diagnostic Tool uses the third-party Jackson library version 2.12.1,
which has known vulnerabilities.
This issue is resolved. The Diagnostic Tool now uses Jackson version 2.13.3.

OGI-3083
When the wrapper.conf file path contains the file and path separator variables,
the Software AG Diagnostic tool does not list bundles which are not in use by
the OSGi profiles of Software AG products.
This issue is resolved.

OGI-3050
The Software AG Common Platform diagnostic tool uses a version of
Apache Log4j 2 that has known vulnerabilities.
This issue is resolved. Apache Log4j is upgraded to the stable
version 2.17.1.

OGI-3028
The Software AG Diagnostic tool does not list bundles which are not in use by
the OSGi profiles of Software AG products.
This issue is resolved. To list these bundles, open a command prompt in Software
AG_directory\common\lib\diagnostic-tool and type the following command:
java -jar bundle-inventory.jar 

OGI-2998
The Software AG Common Platform diagnostic tool uses a version of
Apache Log4j 2 that has known vulnerabilities.
This issue is resolved. Apache Log4j is upgraded to the stable
version 2.16.0.

OGI-2899
When the P2 repository contains two bundles with the same bundle symbolic
name and version, the Software AG Diagnostic tool does not display the duplicate
bundle in the "Extra IUs" section of the validator log file.
This issue is resolved.
 

OGI-2821
The Software AG Diagnostic tool uses a version of the third-party
XNIO library with known security vulnerabilities.
This issue is resolved. XNIO is now upgraded to version 3.8.4
 

OGI-2626
The Software AG Diagnostic tool uses a version of the third-party
Undertow web server with known security vulnerabilities.
This issue is resolved. Undertow is now upgraded to version 2.2.2

OGI-2629
Software AG Common Platform diagnostic tool uses versions of
Log4j with known security vulnerabilities.

The issue is resolved. Logj4 is upgraded to the
stable version 2.13.3.
 

OGI-2681
The Software AG Diagnostic tool can now validate
the OSGi fix metadata of an installation.
 

OGI-2627
Software AG Common Platform diagnostic tool uses versions of
Jackson (i.e. jackson-databind.jar) with known security
vulnerabilities.

The issue is resolved. Jackson databind is upgraded to the
stable version 2.12.1.
 

OGI-2511
The Software AG Diagnostic tool uses a version of the third-party
Undertow web server with known security vulnerabilities.
This issue is resolved. Undertow is now upgraded to version 2.1.3

OGI-2536
The Software AG Diagnostic tool can now validate the OSGi metadata of an installation.
 

OGI-2434
Software AG Common Platform diagnostic tool uses versions of
Log4j with known security
vulnerabilities.

The issue is resolved. Logj4 is upgraded to the
stable version 2.11.2.
 

OGI-2413

Software AG Common Platform diagnostic tool uses versions of
Jackson (i.e. jackson-databind.jar) with known security
vulnerabilities.

The issue is resolved. Jackson databind is upgraded to the
stable version 2.10.1.
 

OGI-2357
Software AG Common Platform diagnostic tool uses versions of
Jackson (i.e. jackson-databind.jar) with known security
vulnerabilities.

The issue is resolved. Jackson databind is upgraded to the
stable version 2.9.10.

installed fix.
