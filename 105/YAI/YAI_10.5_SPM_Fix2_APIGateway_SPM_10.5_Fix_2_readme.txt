1.0 Fix Name

APIGateway SPM 10.5 Fix 2



2.0 Fix ID

YAI_10.5_SPM_Fix2



3.0 Product(s)/Component(s) Affected

This fix affects Software AG APIGateway SPM 10.5.



4.0 Fix Requirements

APIGateway SPM 10.5

For language pack requirements, see the Globalization section of the 
Software AG Empower Product Support website at

https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

APIGateway SPM 10.5 Fix 2 is supported on the same platforms and 
operating systems and with the same JREs as APIGateway SPM 10.5.



6.0 Cautions and Warnings

None.



7.0 Fix Contents
YAI-21643 (YAI_10.5_SPM_Fix2)
Jackson jars in SPM plugin for security vulnerabilities is updated.

YAI-16923 (YAI_10.5_SPM_Fix1)

Unable to update port accessType using a composite template in Command Central.

Unable to update accessType for a port in API Gateway through the Command
Central template.

This issue is resolved. Now, you can update the accessType and IP access type of
a port through Command Central template.



8.0 Installation

Install the fix using the Software AG Update Manager. For instructions, see 
Using the Software AG Update Manager located either in the _documentation 
directory or on the documentation Web site at 
http://documentation.softwareag.com.



9.0 Uninstallation

Uninstall the fix using the Software AG Update Manager. For instructions, see 
Using the Software AG Update Manager.



10.0 Globalization

This fix conforms to the internationalization standards of the webMethods 
product suite and includes support for operation in any country, locale, 
or language as specified in the webMethods Installation Guide. It was not 
tested with non-English configurations and non-ASCII data. However, this fix 
has no globalization impact and can be applied to systems running in any 
supported locale or configuration



10.1 Localization

This fix does not require an updated Language Pack. It might
contain new messages and these messages will appear in English.
