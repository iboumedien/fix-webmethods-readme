7.0 Fix Contents

YAI-21869 (Fix 2)
When you set API Gateway logging explicitly, you receive an error.
The log4j jars are now upgraded with the latest ABE fix.

The issue is resolved.

YAI-19511:

When ABE is used to build API Gateway assets and apigateway.buildLocalRepoOnly 
property is true , then the repository built (as given in the build.source.dir)
is invalid i.e The namespace directory APIGateway is not created.

This issue is now resolved.
