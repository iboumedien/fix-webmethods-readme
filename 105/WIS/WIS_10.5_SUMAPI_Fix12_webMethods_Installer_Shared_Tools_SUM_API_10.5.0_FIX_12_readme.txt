7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

SUM-7026 (WIS_10.5_SUMAPI_Fix12)
IBM webMethods Platform Manager is not able to use IBM account to connect to fix repositories.

SUM-6672 (WIS_10.5_SUMAPI_Fix11)
Add support for IBM carve-out.

SUM-5524 (WIS_10.5_SUMAPI_Fix10)
With IBM webMethods Command Central fix installation sometimes can
result in ArrayIndexOutOfBoundsException.

SUM-5327 (WIS_10.5_SUMAPI_Fix9)
With IBM webMethods Command Central fix uninstallation sometimes can
result in a broken fix profile.

CCE-7157 (WIS_10.5_SUMAPI_Fix9)
With IBM webMethods Command Central the fix repository sometimes can
be updated with wrong credentials and some operations may not be
successful after that.

SUM-4988 (WIS_10.5_SUMAPI_Fix8)
With IBM webMethods Command Central fixes under extended maintenance might
not be visible.

SUM-4706 (WIS_10.5_SUMAPI_Fix7)
With IBM webMethods Command Central already deleted fixes might be
shown as installed.

SUM-4467 (WIS_10.5_SUMAPI_Fix6)
With IBM webMethods Command Central when a fix mirror content is listed,
might not shown all available fixes in it.

SUM-4424 (WIS_10.5_SUMAPI_Fix6)
With IBM webMethods Command Central when importing image repository to
mirror repository, based on existing product repository the operation
might fail with missing fix dependencies.

SUM-4413 (WIS_10.5_SUMAPI_Fix5)
When install fixes with IBM webMethods Command Central always latest fixes
are installed even if older fix versions are selected.

SUM-4340 (WIS_10.5_SUMAPI_Fix4)
With IBM webMethods Command Central when a fix image is created with specific
fixes, the dependencies of these fixes are not added.

SUM-4338 (WIS_10.5_SUMAPI_Fix4)
When exporting an image repo with fixes from the same category
only one of them was exported.

SUM-4267 (WIS_10.5_SUMAPI_Fix3)
With IBM webMethods Command Central when a fix mirror repository is updated
with a single fix, the dependencies might not be retrieved.

SUM-4256 (WIS_10.5_SUMAPI_Fix3)
With IBM webMethods Command Central when getting information about
extended maintenance fixes username and password might be shown in the logs.

SUM-4171 (WIS_10.5_SUMAPI_Fix2)
With IBM webMethods Command Central installation of extended
maintenance fixes might be fail.

SUM-4105 (WIS_10.5_SUMAPI_Fix1)
With IBM webMethods Command Central when create or update mirror
operation is executed, components directory is not deleted.



8.0 Installation

Use IBM webMethods Command Central to install fixes on multiple machines from a central location (9.7 or later),
or IBM webMethods Update Manager to install fixes on one machine at a time. For instructions,
see the documentation at https://docs.webmethods.io.




9.0 Uninstallation

Note: These instructions can only be used to uninstall the most recently installed fix. This action
will revert your installation to the previously installed fix. You cannot use these instructions to
uninstall the previously installed fix.

Uninstall using IBM webMethods Command Central (9.7 or later) or IBM webMethods Update Manager. For instructions, see
the documentation at https://docs.webmethods.io.
