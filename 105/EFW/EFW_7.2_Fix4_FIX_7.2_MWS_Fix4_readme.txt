7.0 Fix Contents

WF-34119

Fix includes security updates with no new features to existing user-facing
functionalities.

EFF-444 (EFW_7.2_Fix3)

In IBM webMethods FIX Module, the copyright information is updated.

This issue is now resolved.

EFF-360 (EFW_7.2_Fix2)
The copyright information in the About page of My webMethods is
incorrect.

This issue is resolved. The copyright information in the About
page of My webMethods is updated to display the correct
information. 

EFF-347 (EFW_7.2_Fix1)

In IBM webMethods FIX Module 7.2, system returns error 
while creating a secure connection between MWS and 
FIX Module. 

In IBM webMethods FIX Module 7.2, system fails to establish 
a https connection between MWS and FIX Module and returns 
error "Error retreiving appia connection details from FIX 
module. Check the integration server and FIX Module 
configuraqtion". This issue occurs because FIX Module 
is handling secure connection request incorrectly. 

This issue is resolved and a secure connection can be 
established between MWS and FIX Module.



8.0 Installation

Use Command Central to install fixes on multiple machines from a central

one machine at a time. For instructions,
see the documentation at https://docs.webmethods.io/.



9.0 Uninstallation

Uninstall using Command Central (9.7 or later) or Update Manager.
