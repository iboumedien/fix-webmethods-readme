7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-4676 (TPP_10.15_BaseSecurity_Fix2)

webMethods Shared Libraries contains an outdated version of the third party ESAPI library,  
which has known security vulnerabilities.

The issue is resolved. The following library has been updated:  
The ESAPI library is updated from version 2.5.3.1 to 2.6.0.0.

TPS-2516 (TPP_10.15_BaseSecurity_Fix1)

webMethods Shared Libraries contains the third party library ESAPI with version 2.4.0.0,
which has known security vulnerabilities.

The issue is resolved. The ESAPI library is now updated to version 2.5.3.1.
