1.0 Fix Name

Metering Server 10.15 Fix 4



2.0 Fix ID

MTA_10.15_Server_Fix4


3.0 Product(s)/Component(s) Affected


This fix affects wM Metering 10.15.



4.0 Requirements

This fix requires Metering Server 10.15. 
Metering Server 10.15 Fix 4 
must be installed together with SJP.Java64.LNXAMD64_10.15.0_Fix354,wMFix.SJP.Ja 
if the products for these fixes are installed on the system. 

Fixes are cumulative. Therefore, for each of the listed fixes, you can install
the specified fix or later.


5.0 Platform Support

Same platforms as product release.


6.0 Cautions and Warnings

All product profiles that run on top of OSGi runtimes are stopped by
the IBM webMethods Update Manager and will automatically be started
after the session completes.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes
for the release.

MTA-1828
After installing CJP_10.15_Fix354 that upgrades to Java 17, the Metering Server on premises
does not display usage data.
This issue is resolved. The Metering Server on premises is now compatible with Java 17.

MTA-1791
After installing CJP_10.15_Fix354 that upgrades to Java 17, the Metering Server on premises
does not display usage data.
This issue is resolved. The Metering Server on premises is now compatible with Java 17.

MTA-992
The App Switcher menu is now removed from the graphic user interface of the Metering Local Server.



MTA-742
Software AG Metering contains an outdated tooltip text.
This issue is resolved.

MTA-854
The Software AG Metering Server uses the third-party Liquibase library v4.4.3,
which has ? security vulnerability CVE-2022-0839.
This issue is resolved. The Metering Server now uses the stable version
Liquibase 4.19.0.

MTA-806
The Software AG Metering Server web UI may get stuck loading indefinitely.
This issue is resolved.

MTA-864
On the Usage Report page of the Software AG Metering UI, the Customer and Tenant
drop-down arrows appear double.
This issue is resolved.



8.0 Installation

Use Command Central to install fixes on multiple machines from a central
location (9.7 or later products), or Software AG Update Manager to install
fixes on one machine at a time. For instructions, see the documentation at
http://documentation.softwareag.com.

Note: These instructions can only be used to uninstall the most recently
