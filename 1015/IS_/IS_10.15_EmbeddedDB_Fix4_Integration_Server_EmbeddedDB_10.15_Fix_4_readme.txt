7.0 Fix Contents

PIE-94643
Internal changes to support OAuth usage by layered products.  

PIE-85102 (IS_10.15_EmbeddedDB_Fix3)
Migrating tables for Embedded Database from Integration Server 10.11 to 10.15 
fails with the error: 'path not found for 10.11.fix3 POSTGRESQL to latest'.

This issue is now resolved.

PIE-80727 (IS_10.15_EmbeddedDB_Fix2)
Database components in Integration Server fail to work after migrating 
Integration Server from 10.3 to 10.11 or higher. 

When migrating Integration Server from 10.3 to 10.11 or higher, the embedded database is 
not updated with new tables. Hence, the database components fail to work. 

The issue is resolved now.
