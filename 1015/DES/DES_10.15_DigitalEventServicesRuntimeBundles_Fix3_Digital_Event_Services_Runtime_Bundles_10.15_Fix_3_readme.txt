7.0 Fix Contents


DEV-2188 (DEV_10.15_Digital Event Services Runtime Bundles_Fix3)

The Digital Event Services Shared Bundles 10.15 Fix 2 installation fails because of a missing dependency to webMethods Shared Bundles DataModelling 10.15 Fix 4. 

The issue is resolved.

TPS-4759 (DEV_10.15_Digital Event Services Runtime Bundles_Fix2)

webMethods Shared Libraries contains the third party SLF4J and Log4j libraries,
which have known security vulnerabilities.

The issue is resolved.


8.0 Installation

Use Command Central to install fixes on multiple machines from a central 
location (9.7 or later products), or webMethods Update Manager to install 
fixes on one machine at a time. For instructions, see the documentation at
http://documentation.softwareag.com.



9.0 Uninstallation

Note: These instructions can only be used to uninstall the most recently 
installed fix. This action will revert your installation to the previously 
installed fix. You cannot apply these instructions to the previously 
installed fix.

Uninstall using Command Central (9.7 or later) or webMethods Update Manager. 
For instructions, see the documentation at
http://documentation.softwareag.com.
