7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

ABE-619 (ABE_10.15_Fix2)

Asset Build Environment uses an outdated version of the third-party component JGit, which has known
security vulnerabilities.

The issue is resolved. The JGit jar is migrated to the stable version 7.2.1.

ABE-628 (ABE_10.15_Fix2)

Asset Build Environment uses an outdated version of the third-party component Apache Ant, which has
known security vulnerabilities.

The issue is resolved. The Apache Ant component is migrated to the stable version 1.10.12.2.

ABE-380 (ABE_10.15_Fix1)

Asset Build Environment uses an outdated version of the
third party component JGit, which has known security vulnerabilities.

The issue is resolved. The JGit jar is migrated to the stable version 6.7.0.
