7.0 Fix Contents

In addition to including resolutions to product defects, a fix may include 
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters.
Review the fix contents carefully before applying the fix.

This fix is cumulative, which means that it is an aggregate of previous fixes 
for the release.

ASH-572

Fix includes security updates with no new features to existing user-facing
functionalities.

ASH-551 (ASH_10.1_Fix3)

In the IBM webMethods module for AS4AU, the copyright information is updated.

This issue is now resolved.

ASH-386 (ASH_10.1_Fix2)

In Agreement Update Handler, documents generated using IBM webMethods Module for
AS4AU message generation services contain incorrect namespace value for
"xmlns:au" attribute in "xs:schema" element.

This issue is resolved.

In Agreement Update Handler, agreement update fails to update the certificate if
the Certificate Update Request consists only a leaf certificate. The error, 
"Bad certificate chain for usage decrypt: Certificate chain verification failed"
appears.
