1.0 Fix Name

webMethods Shared Bundles DataDirect 10.15 Fix 6



2.0 Fix ID

DDJ_10.15_DataDirect_Fix6



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.15 webMethods products.



4.0 Requirements and Dependencies

None.



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-4714 (DDJ_10.15_DataDirect_Fix6)

webMethods Shared Bundles DataDirect 10.15 fixes have previously updated the DataDirect drivers only if product profiles were present. 
This caused issues in side-by-side or profile-less installations.

This fix ensures that the DataDirect drivers are always updated, regardless of whether product profiles exist.

TPS-4617 (DDJ_10.15_DataDirect_Fix5)

webMethods Shared Bundles DataDirect 10.15 contains Ð° redundant Jackson Databind service
used internally within the DataDirect drivers.

The issue is resolved. The service has been removed from the drivers.

TPS-4560 (DDJ_10.15_DataDirect_Fix4)

webMethods Shared Bundles DataDirect 10.15 contains outdated DataDirect drivers.

The issue is resolved. The following drivers have been updated:
The DataDirect Oracle driver has been upgraded from version 6.0.0.1165 to version 6.0.0.1408.
The DataDirect SQL Server driver has been upgraded from version 6.0.0.1282 to version 6.0.0.1642.
The DataDirect DB2 driver has been upgraded from version 5.1.4.0416 to version 5.1.4.0455.
The DataDirect MySQL driver has been upgraded from version 5.1.4.0266 to version 5.1.4.0315.

TPS-3280 (DDJ_10.15_DataDirect_Fix3)

webMethods Shared Bundles DataDirect 10.15 contains an outdated Oracle driver.

The issue is resolved. The DataDirect Oracle driver has been upgraded to version 6.0.0.1165.

TPS-2326 (DDJ_10.15_DataDirect_Fix2)

webMethods Shared Bundles DataDirect 10.15 contains a faulty Oracle driver, which causes connection problems.

The issue is resolved. The DataDirect libraries are now updated to version 6.0.0.1098

TPS-2144 (DDJ_10.15_DataDirect_Fix1)

webMethods Shared Bundles DataDirect 10.15 contains a faulty Oracle driver, which causes connection problems.

The issue is resolved. The DataDirect libraries are now updated to version 6.0.0.977.
