7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include
enhancements to existing functionality and features. Any of these items can 
result in new or changed built-in services, APIs, or configuration parameters. 
Review the fix contents carefully before applying the fix.

This fix is cumulative, which means that it is an aggregate of previous fixes 
for the release.

OGI-3962
The Software AG Diagnostic Tool uses version 2.15.2 of the third-party jackson-databind,
jackson-core, and jackson-annotations libraries, which have known vulnerabilities.
This issue is resolved. The Diagnostic Tool now uses version 2.16.1 of these libraries.

OGI-3841
The Software AG Diagnostic Tool uses version 2.14.0-rc1 of the third-party jackson-databind,
jackson-core, and jackson-annotations libraries, which have known vulnerabilities.
This issue is resolved. The Diagnostic Tool now uses version 2.15.2 of these libraries.

OGI-3610
The Installation Validator does not generate correct fix-level diagnostic
information about the Software AG Common Platform and the OSGi profiles
of Software AG products.
This issue is resolved.

OGI-3520
The Software AG Diagnostic Tool throws a NullPointerException error
if any of the default folders in the Software AG installation is missing.
The issue is resolved.

OGI-3495
The Installation Validator does not generate a diagnostic information file about
the Software AG Common Platform and the OSGi profiles of Software AG products
in a readable format.
This issue is resolved.
