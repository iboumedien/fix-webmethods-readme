6.0 Cautions and Warnings

None


7.0 Fix Contents

MWS-32666 (MWS_10.15_SPM_Fix2)
When creating My webMetods Server instance using the Command Central
web user interface, the drop-down menu for selecting the database type
does not include PostgreSQL, MySQL EE, and MySQL CE databases, although 
all three types are supported.

The issue is resolved. The web interfaces for creating My webMethods 
Server instances are enhanced to display all supported database types.


MWS-31743 (MWS_10.15_SPM_Fix1)
My webMethods Server nodes might report an "Invalid content was found..."
in the Command Central graphical user interface. This issue occurs when
the mws.db.xml for My webMethods Server contains the RETRYCOUNT and 
RETRYDELAY elements that configure the database connection retries 
number and delay period.

The issue is resolved.

9.0 Uninstallation

These instructions can only be used to uninstall the most recently
installed fix. This action will revert your installation to the
previously installed fix. You cannot use these instructions to uninstall
the previously installed fix.

Uninstall using Software AG Update Manager. For instructions, see the
documentation at http://documentation.softwareag.com.

You can only uninstall one My webMethods Server fix at a time. Make
sure you backup your server settings before uninstalling My webMethods
Server fixes.


10.0 Globalization

This fix conforms to the internationalization standards of the
webMethods product suite and includes support for operation in
any country, locale, or language as specified in the Installing
webMethods Products guide. It was not tested with non-English
configurations and non-ASCII data. However, this fix has no
globalization impact and can be applied to systems running in
any supported locale or configuration.

9.1 Localization

This fix does not require an updated language pack. It might
contain new messages and these messages will appear in English.
