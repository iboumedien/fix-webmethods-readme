8.0 Installation

Install the fix using the Software AG Update Manager.
For instructions, see Using the Software AG Update Manager
located either in the _documentation directory or on the 
documentation Web site at http://documentation.softwareag.com.


9.0 Uninstallation

These instructions can only be used to uninstall the most recently
installed fix. This action will revert your installation to the
previously installed fix. You cannot use these instructions to uninstall
the previously installed fix.

Uninstall using Software AG Update Manager. For instructions, see the
documentation at http://documentation.softwareag.com.

To uninstall Trading Networks Portal Platform Manager Plug-in 10.15 Fix 1
(wMFix.TNPortal_10.15_SPM_Fix1) from Infrastructure.Platform Manager 
Plug-ins.Trading Networks.server 10.15, you must first uninstall Shared 
Bundles Glassfish 10.15 Fix 1(wMFix.TPS.Glassfish_10.15.0.0001).
