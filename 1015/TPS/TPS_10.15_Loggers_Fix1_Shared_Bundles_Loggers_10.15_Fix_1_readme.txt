TPS-4760 (TPS_10.15_Loggers_Fix1)

webMethods Shared Bundles contains outdated versions of the third party SLF4J and Log4j libraries,
which have known security vulnerabilities.

The issue is resolved. The following libraries have been updated:
The slf4j-api library is updated from version 1.7.30 to version 2.0.16.
The slf4j-jdk14 library is updated from version 1.7.30 to version 2.0.16.
The slf4j-ext library is updated from version 1.7.30 to version 2.0.16.
The log4j-12-api library is updated from version 2.17.1 to version 2.25.3.
The log4j-api library is updated from version 2.17.1 to version 2.25.3.
The log4j-core library is updated from version 2.17.1 to version 2.25.3.
The log4j-jcl library is updated from version 2.17.1 to version 2.25.3.
The log4j-jul library is updated from version 2.17.1 to version 2.25.3.
The log4j-slf4j-impl library is updated from version 2.17.1 to version 2.25.3.
