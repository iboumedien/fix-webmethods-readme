7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is not cumulative. This means that installing the latest fix only brings the latest version 
and does not contain the changes from the previous fixes.



TPS-2092 (TPS_10.15_Tanuki_Fix2)

webMethods Shared Bundles contains the third party library Tanuki with version 3.5.50, 
which causes unexpected issues.

The issue is resolved. The Tanuki library is now updated to version 3.5.53

TPS-1918 (TPS_10.15_Tanuki_Fix1)

[NUM-18785] Tanuki wrapper 3.5.48 introduced a sporadic issue affecting Linux console processes running as deamons, 
which is a common use for UM Server. 

Unfortunately this issue is still present in 3.5.50 and started exhibiting itself in tests run by other teams. UM will switch back to the 
last version of Tanuki which did not have this issue - v3.5.43. To fulfill this purpose TPS fix will bring back Tanuki v3.5.43 to common/lib 
and common/bin directories.
