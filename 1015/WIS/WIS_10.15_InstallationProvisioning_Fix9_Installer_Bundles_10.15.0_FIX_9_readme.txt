7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

WIR-15180 (WIS_10.15_InstallationProvisioning_Fix9)
When using IBM webMethods Command Central to install products, product dependencies might not be
collected correctly if the product dependency itself is targeting only a subset of the supported
platforms by the selected product.

WIR-15013 (WIS_10.15_InstallationProvisioning_Fix7)
IBM webMethods Common Platform uses Apache Tomcat, which has security vulnerabilities. Now the version is updated to secure one.

WIR-15013 (WIS_10.15_InstallationProvisioning_Fix5)
IBM webMethods Installer Shared component Provisioning Application is not able to use IBM account for authentication and entitlement check.

WIR-14734 (WIS_10.15_InstallationProvisioning_Fix3)
Add support for IBM carve-out

SUM-5979 (WIS_10.15_InstallationProvisioning_Fix2)
The user password is displayed in the logs.
The issue is resolved by masking the password.

WIR-14516 (WIS_10.15_InstallationProvisioning_Fix1)
When using Command Central to import an image to the product repository,
the operation might fail with the error message "No Match Found".
To resolve this issue, install the fix, create a new image using
Command Central, and import it into the product repository.




8.0 Installation

Use IBM webMethods Command Central to install fixes on multiple machines from a central location (9.7 or later),
or IBM webMethods Update Manager to install fixes on one machine at a time. For instructions,
see the documentation at https://docs.webmethods.io.
