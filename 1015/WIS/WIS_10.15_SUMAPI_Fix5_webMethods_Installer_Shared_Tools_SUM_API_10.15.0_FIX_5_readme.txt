7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

SUM-7026 (WIS_10.15_SUMAPI_Fix5)
IBM webMethods Platform Manager is not able to use IBM account to connect to fix repositories.

SUM-6672 (WIS_10.15_SUMAPI_Fix4)
Add support for IBM carve-out.

SUM-6333 (WIS_10.15_SUMAPI_Fix3)
The IBM webMethods Command Central fix mirror operation shows a
warning message in the log files. This warning message does not
affect the mirror operation and is suppressed.

CCE-8210 (WIS_10.15_SUMAPI_Fix2)
With IBM webMethods Command Central a fix might be excluded from resolving
using a property 'com.softwareag.plm.sum.cc.skip.fix.dependency'

SUM-5524 (WIS_10.15_SUMAPI_Fix1)
With IBM webMethods Command Central fix installation sometimes can
result in ArrayIndexOutOfBoundsException.

SUM-5327 (WIS_10.15_SUMAPI_Fix1)
With IBM webMethods Command Central fix uninstallation sometimes can
result in a broken fix profile.




8.0 Installation

Use IBM webMethods Command Central to install fixes on multiple machines from a central location (9.7 or later),
or IBM webMethods Update Manager to install fixes on one machine at a time. For instructions,
see the documentation at https://docs.webmethods.io.




9.0 Uninstallation

Note: These instructions can only be used to uninstall the most recently installed fix. This action
will revert your installation to the previously installed fix. You cannot use these instructions to
uninstall the previously installed fix.

Uninstall using IBM webMethods Command Central (9.7 or later) or IBM webMethods Update Manager. For instructions, see
the documentation at https://docs.webmethods.io.
