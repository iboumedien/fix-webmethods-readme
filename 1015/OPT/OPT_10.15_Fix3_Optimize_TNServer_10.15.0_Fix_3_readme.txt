1.0 Fix Name

webMethods Optimize InfrastructureDC 10.15.0 Fix 3

2.0 Fix ID

WID_10.15.0_Fix3_InfrastructureDC


3.0 Product(s)/Component(s) Affected

This fix affects:

- Infrastructure Data Collector 10.15


4.0 Requirements and Dependencies

None

Fixes are cumulative. Therefore, for each of the listed fixes, you can install the specified fix or 
later.

5.0 Platform Support

Same platforms as product release.

6.0 Cautions and Warnings

Attempting to install this fix in a My webMethods Server environment that is configured to use an
embedded Derby database fails. You can find the error details about the fix installation failure in
the IBM webMethods Update Manager log file.


7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

WID-2365
When Infrastructure Data Collector monitors Integration Server components, error messages appear in the server log.
The issue is resolved.

WID-2387
The WmARTExtDC package causes error messages to appear in the server log. 
The issue is resolved.

WID-2321
Optimize for Infrastructure cannot discover Terracotta BigMemory Max 4.4.0.12.
The issue is resolved.

WID-2331
In Optimize for Infrastructure, the <SAG_Install_Dir>/InfrastructureDC/bin/migrate/migrate.sh utility throws an exception.
The issue is resolved.

WID-2180 (WID_10.15_Fix2)
In webMethods Optimize, after you discover an ART adapter connection, an "Encountered exception while handling adapter notfication: java.lang.NullPointerException" error appears in the server.log file. 
The issue is resolved.

WID-1989 (WID_10.15_Fix1)
Migration of InfrastructureDC from lower version to 10.15 fails
The issue is resolved.

WID-2089 (WID_10.15_Fix1)
InfrastructureDC migration utility fails on startup due to NullPointerException
The issue is resolved.

WID-2091 (WID_10.15_Fix1)
DISCOVERY of IS SERVERS Fails
The issue is resolved. 

WID-2007 (WID_10.15_Fix1)
Optimize creates Rule Violations for no reason until WmISExtDC package is restarted.
The issue is resolved.

WID-2093 (WID_10.15_Fix1)
Failed to discover IS host with underscore
The issue is resolved.

WID-2011 (WID_10.15_Fix1)
Discovery failing for some IS servers.
The issue is resolved.

WID-2137 (WID_10.15_Fix1)
webMethods Optimize for Infrastructure cannot connect to an SSL-enabled Universal Messaging server when deploying the environment.
The issue is resolved.
