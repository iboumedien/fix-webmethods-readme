7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

MWS-30461 (DES_10.7_CAD_Fix3)
A security issue with the third-party library Hibernate Validator was discovered in Designer.

The issue is resolved. The Hibernate Validator library is upgraded to a more secure version.


MWS-32707 (DES_10.7_CAD_Fix3)
A security issue with the third-party component Apache Jackrabbit was discovered in My webMethods
Server. 

The issue is resolved. The Apache Jackrabbit component is upgraded to a more secure
version. For the fix to take effect, you must install the corresponding My webMethods 
Server fix.


WMC-5294 (DES_10.7_CAD_Fix3)
Software AG Designer might wire to two versions of the third-party component Commons-fileupload
when bundles with different symbolic names are available in the installation.

The issue is resolved. Designer now wires to the upgraded version of the library bundle.


WMC-5180 (DES_10.7_CAD_Fix2)
A security issue with the third-party library Commons-fileupload was discovered in Designer.

The issue is resolved. The Commons-fileupload library is upgraded to the more secure version 1.5.


WMC-5188 (DES_10.7_CAD_Fix2)
A security issue with the third-party library chart.js was discovered in My webMethods Server.

The issue is resolved. For the fix to take effect, you must install the corresponding My webMethods 
Server fix.


WMC-5223 (DES_10.7_CAD_Fix2)
Editing large XHTML files in Software AG Designer might require a significant amount of time.

The issue is resolved.


MWS-27467 (DES_10.7_CAD_Fix1)
Some of the UI labels on the task voting user interfaces are not capitalized consistently and have
duplicate entries.

The issue is resolved.


WTN-5387 (DES_10.7_CAD_Fix1)
When editing tasks created with earlier versions, Software AG Designer might attempt to add voting
details to non-votable task types and log a NullPointerException error message.

The issue is resolved.


WTN-5402 (DES_10.7_CAD_Fix1)
webMethods Task Engine might fail to reindex the task data for HPSTRA-enabled tasks. This issue
occurs when the IS document type that defines the task business data includes one or more fields
of type List (array).

The issue is resolved. Task Engine now provides support for HPSTRA-enabled tasks that include
business data from an IS document type reference with List (array) fields. For this fix to take
effect, you must also install the corresponding Designer fix. For existing task projects, you
must also perform the following manual steps: 1. Open the task type in Designer and remove all
List fields from IS document types you want to index in Elasticsearch from the task business
ata.  2. Add the fields again and rebuild the project. 3. Redeploy the task type to My webMethods
Server.


WTN-5650 (DES_10.7_CAD_Fix1)
When creating tasks on the UI Development perspective, Software AG Designer might generate the
taskRules.xml file incorrectly, causing task assignments to fail with a NullPointerException
error message at runtime.

The issue is resolved. For the fix to take effect, you must install the corresponding My webMethods 
Server fix.

10.0 Globalization

This fix conforms to the internationalization standards of the webMethods product suite and includes 
support for operation in any country, locale, or language as specified in the Installing webMethods 
Products guide. It was not tested with non-English configurations and non-ASCII data. However, this 
fix has no globalization impact and can be applied to systems running in any supported locale or 
configuration.
