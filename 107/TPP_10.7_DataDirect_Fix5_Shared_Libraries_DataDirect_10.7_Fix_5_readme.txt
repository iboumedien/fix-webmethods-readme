1.0 Fix Name

webMethods Shared Libraries DataDirect 10.7 Fix 5



2.0 Fix ID

TPP_10.7_DataDirect_Fix5



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.7 webMethods products.



4.0 Requirements and Dependencies

Fixes are cumulative. Therefore, for each of the listed fixes, you can install the specified fix or 
later.



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-4580 (TPP_10.7_DataDirect_Fix5)

webMethods Shared Libraries DataDirect 10.7 contains outdated DataDirect drivers.

The issue is resolved. The following drivers have been updated:
The DataDirect Oracle driver has been upgraded from version 6.0.0.0977 to version 6.0.0.1408.
The DataDirect SQL Server driver has been upgraded from version 6.0.0.1071 to version 6.0.0.1642.
The DataDirect DB2 driver has been upgraded from version 5.1.4.336 to version 5.1.4.0455.
The DataDirect MySQL driver has been upgraded from version 5.1.4.266 to version 5.1.4.0315.

TPS-2121 (TPP_10.7_DataDirect_Fix4)

webMethods Shared Libraries DataDirect 10.7 contains corrupted DataDirect driver, that is causing failure with WDA [webMethods Database Configuration]
WDA-1886: "Not able to connect through DB Configurator"
Issue is resolved with the following change:
 * The DataDirect library is updated to version 6.0.0.97

TPS-1557 (TPP_10.7_DataDirect_Fix3)

webMethods Shared Libraries DataDirect 10.7 contains corrupted DataDirect driver, that is causing failure in starting MWS
TPS-1198: "MWS10.7 unable to work with Azure SQL DB"
Issue is resolved with the following change:
 * The DataDirect library is updated to version 6.0.0.8

 
TPS-1354 (TPP_10.7_DataDirect_Fix2)

webMethods Shared Libraries DataDirect 10.7 contains corrupted DataDirect driver, that is causing failure in starting MWS
Issue is resolved with the following change:
 * The DataDirect library is updated to version 6.0.0.6


TPS-1135 (TPP_10.7_DataDirect_Fix1)

The Shared Libraries 10.7 DataDirect Fix 1 fix addresses driver version security issues with the following change:
 * The DataDirect library is updated to version 6.0.0.5
