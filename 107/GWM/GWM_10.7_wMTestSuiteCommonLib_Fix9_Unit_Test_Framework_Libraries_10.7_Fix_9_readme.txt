7.0 Fix Contents

GWM-744

Unit Test Framework fails with the following error when there are
'com.wm.data.BasicIDataCodable' records in the service response. 
Particularly, if the service involves Trading Networks and SAP adapter:
 
Expected IData object but encountered.: expected <IData> but was
<com.wm.util.Values>
 
During the output comparison, the comparator service in 
Unit Test Framework fails with the java.lang.AssertionError error 
if the response from a service under test contains 
'com.wm.data.BasicIDataCodable' records.

This issue is now resolved.
