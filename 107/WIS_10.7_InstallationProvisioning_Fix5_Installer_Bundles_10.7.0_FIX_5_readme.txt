7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.

WIR-15209 (WIS_10.7_InstallationProvisioning_Fix5)
IBM webMethods Installer Shared component Provisioning Application is not able to use IBM account for authentication and entitlement check.

WIR-14095 (WIS_10.7_InstallationProvisioning_Fix3)
Installation of webMethods Installer Shared Tools InstallationProvisioning 10.7 Fix 2
might fail, because fix dependencies are not explicitly declared.

Required dependencies are defined.

The issue is resolved. 


WIR-14043
With IBM webMethods Commmand Central installation of Integration Server might fail.
The issue is resolved.


WIR-13166
With IBM webMethods Command Central running multiple product repository validations in parallel might result 
in sporadical failure.

The issue is resolved. 

WIR-13110 (WIS_10.7_InstallationProvisioning_Fix2)
With IBM webMethods Command Central installation of fixes might fail, because of certificate mismatch.

The issue is resolved.

WIR-13031 (WIS_10.7_InstallationProvisioning_Fix1)
With IBM webMethods Central some products might not be reported as
installed.

The issue is resolved.
