1.0 Fix Name

webMethods Shared Libraries SpringFramework 10.7 Fix 3



2.0 Fix ID

TPP_10.7_SpringFramework_Fix3



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.7 webMethods products.



4.0 Requirements and Dependencies

Fixes are cumulative. Therefore, for each of the listed fixes, you can install the specified fix or 
later.

For language pack requirements, see the Globalization section of the Software AG Empower Product 
Support website at https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-2325 (TPP_10.7_SpringFramework_Fix3)

webMethods Shared Libraries contains outdated versions of the third party Spring Framework libraries.

The issue is resolved. The following Spring Framework libraries are now updated:
The spring-aop library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-beans library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-context library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-core library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-expression library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-jdbc library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-jms library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-test library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-transaction library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-orm library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-web library is updated from version 5.3.18.1 to version 5.3.27.1.
The spring-context-support library is updated from version 5.3.18.1 to version 5.3.27.1.

> spring-beans is updated from 5.3.14.1 to 5.3.18.1
> spring-context is updated from 5.3.14.1 to 5.3.18.1
> spring-core is updated from 5.3.14.1 to 5.3.18.1
> spring-expression is updated from 5.3.14.1 to 5.3.18.1
> spring-jdbc is updated from 5.3.14.1 to 5.3.18.1
> spring-jms is updated from 5.3.14.1 to 5.3.18.1
> spring-test is updated from 5.3.14.1 to 5.3.18.1
> spring-transaction is updated from 5.3.14.1 to 5.3.18.1
> spring-orm is updated from 5.3.14.1 to 5.3.18.1
> spring-web is updated from 5.3.14.1 to 5.3.18.1
> spring-webmvc is updated from 5.3.14.1 to 5.3.18.1
> spring-context-support is updated from 5.3.14.1 to 5.3.18.1
