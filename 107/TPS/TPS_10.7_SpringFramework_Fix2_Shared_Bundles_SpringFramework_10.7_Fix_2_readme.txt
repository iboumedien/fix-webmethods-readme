7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-1624 (TPS_10.7_SpringFramework_Fix2)

CCE team have requested an explicit dependency on TPS SpringFramework fix to SPM fixes do to issues where SPM fails to come online. Shared Bundles TPS SpringFramework fix now has a dependency to SPM fix.

TPS-1461 (TPS_10.7_SpringFramework_Fix1)

webMethods Shared Bundles contain the third party SpringFramework with version 5.2.4.1, which has the Spring4Shell vulnerability associated with it.

The issue is resolved. The following Spring libraries have been updated to the newest safe versions:
> spring-aop is updated from 5.2.4.1 to 5.3.18.1
> spring-beans is updated from 5.2.4.1 to 5.3.18.1
> spring-context is updated from 5.2.4.1 to 5.3.18.1
> spring-core is updated from 5.2.4.1 to 5.3.18.1
> spring-expression is updated from 5.2.4.1 to 5.3.18.1
> spring-jdbc is updated from 5.2.4.1 to 5.3.18.1
> spring-jms is updated from 5.2.4.1 to 5.3.18.1
> spring-test is updated from 5.2.4.1 to 5.3.18.1
> spring-transaction is updated from 5.2.4.1 to 5.3.18.1
> spring-orm is updated from 5.2.4.1 to 5.3.18.1
> spring-web is updated from 5.2.4.1 to 5.3.18.1
> spring-webmvc is updated from 5.2.4.1 to 5.3.18.1
> spring-context-support is updated from 5.2.4.1 to 5.3.18.1
