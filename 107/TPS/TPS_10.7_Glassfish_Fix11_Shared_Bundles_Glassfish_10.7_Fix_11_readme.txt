1.0 Fix Name

webMethods Shared Bundles Glassfish 10.7 Fix 11



2.0 Fix ID

TPS_10.7_Glassfish_Fix11



3.0 Product(s)/Component(s) Affected

This fix affects the common extended libraries installed with the 10.7 webMethods products.



4.0 Requirements and Dependencies

Fixes are cumulative. Therefore, for each of the listed fixes, you can install the specified fix or 
later.

For language pack requirements, see the Globalization section of the Software AG Empower Product 
Support website at https://empower.softwareag.com/Products/Globalization/default.asp



5.0 Platform Support

Same platforms as product release.



6.0 Cautions and Warnings

None.



7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes for the release.


TPS-2563 (TPS_10.7_Glassfish_Fix11)

webMethods Shared Libraries contains the outdated third party library Jackson Databind with version 2.15.2.

The issue is resolved. The following Jackson libraries are now updated:
The SnakeYAML library is now updated from version 2.0.0 to version 2.2.0.
The jackson-core library is updated from version 2.15.2 to version 2.15.3.
The jackson-annotations library is updated from version 2.15.2 to version 2.15.3.
The jackson-databind library is updated from version 2.15.2 to version 2.15.3.
The jackson-dataformat-yaml library is updated from 2.15.2 to version 2.15.3.
The jackson-dataformat-cbor library is updated from version 2.15.2 to version 2.15.3.
The jackson-module-jaxb-annotations library is updated from version 2.15.2 to version 2.15.3.
The jackson-datatype-jsr310 library is updated from version 2.15.2 to version 2.15.3.
The jackson-jaxrs-base library is updated from version 2.15.2 to version 2.15.3.
The jackson-jaxrs-json-provider library is updated from version 2.15.2 to version 2.15.3.

TPS-2342 (TPS_10.7_Glassfish_Fix10)

webMethods Shared Bundles contains the third party library Jackson Databind with version 2.15.0-rc3, which causes unexpected issues.

The issue is resolved. The following Jackson libraries are now updated:
The jackson-core library is updated from version 2.15.0-rc3 to version 2.15.2.
The jackson-annotations library is updated from version 2.15.0-rc3 to version 2.15.2.
The jackson-databind library is updated from version 2.15.0-rc3 to version 2.15.2.
The jackson-dataformat-yaml library is updated from 2.15.0-rc3 to version 2.15.2.
The jackson-dataformat-cbor library is updated from version 2.15.0-rc3 to version 2.15.2.
The jackson-module-jaxb-annotations library is updated from version 2.15.0-rc3 to version 2.15.2.
The jackson-jaxrs-base library is updated from version 2.15.0-rc3 to version 2.15.2.
The jackson-jaxrs-json-provider library is updated from version 2.15.0-rc3 to version 2.15.2.
The fastdoubleparser library has been removed as it is no longer required by jackson-core

TPS-2323 (TPS_10.7_GlassFish_Fix9)

webMethods Shared Bundles contains the third party library SnakeYAML with version 1.33 which has known security vulnerabilities and outdated versions of Jackson libraries.

The issue is resolved. The following libraries are now updated:

The jackson-core library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.
The jackson-annotations library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.
The jackson-databind library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.
The jackson-dataformat-yaml library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.
The jackson-dataformat-cbor library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.
The jackson-module-jaxb-annotations library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.
The jackson-jaxrs-base library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.
The jackson-jaxrs-json-provider library is updated from version 2.14.0-rc2 to version 2.15.0-rc3.

TPS-2202 (TPS_10.7_GlassFish_Fix8)

The issue is resolved. The Jettison library is now updated to version 1.5.3.15-v0011.

TPS-2061 (TPS_10.7_GlassFish_Fix6)

webMethods Shared Bundles contains the third party woodstox-core with version 6.4.0 which breaks warnings-as-errors builds due to missing Resolution.OPTIONAL enumeration

The issue is resolved. The following library has been updated and the package has been added:
> woodstox-core is updated from v6.4.0 to v6.5.0

TPS-1907 (TPS_10.7_GlassFish_Fix5)

wMFix.MFTServer_10.7.0.0014-0002 fix fails to install.

The issue is resolved.
The following library has been updated:
> jakarta-mail is updated from v1.6.7 to v1.6.7.1

webMethods Shared Bundles contains the third party woodstox-core with version 6.1.1, which has known security vulnerabilities:
[CVE-2022-40156], [CVE-2022-40156], [CVE-2022-40155], [CVE-2022-40154], [CVE-2022-40153].

The issue is resolved. The following library has been updated:

> stax2-api is updated from v4.2.0 to v4.2.1

TPS-1551 (TPS_10.7_GlassFish_Fix4)

webMethods Shared Bundles contains the third-party Jackson Databind with version 2.13.2.2 and SnakeYAML with version 1.30, which have known security vulnerabilities:
[CVE-2022-42003], [CVE-2022-42004], [CVE-2022-25857].

The issue is resolved. Following Jackson libraries have been updated:
> jackson-core is updated from v2.12.3 to v2.14.0-rc2
> jackson-annotations is updated from v2.12.3 to v2.14.0-rc2
> jackson-databind is updated from v2.12.3 to v2.14.0-rc2
> jackson-dataformat-yaml is updated from v2.12.3 to v2.14.0-rc2
> jackson-dataformat-cbor is updated from v2.12.3 to v2.14.0-rc2
> jackson-module-jaxb-annotations is updated from v2.12.3 to v2.14.0-rc2
> jackson-jaxrs-base is updated from v2.12.3 to v2.14.0-rc2
> jackson-jaxrs-json-provider is updated from v2.12.3 to v2.14.0-rc2
> snakeyaml is updated from v1.27.0 to v1.33.0

The following library has also been updated to allow support for OAuth in POP3:
> jakarta-mail is updated from v1.6.3 to v1.6.7

TPS-1335 (TPS_10.7_Glassfish_Fix3)

webMethods Shared Bundles contain the third parties Jakarta EL with version v3.0.2 and jersey2-common with version 2.29.1, which have known security vulnerabilities.

The issue has been resolved. The following libraries have been updated:

> Jakarta EL has been updated from v3.0.2 to v3.0.4

> jersey2-container-servlet-core.jar updated from 2.29.1 to 2.34.0
> jersey2-container-servlet.jar updated from 2.29.1 to 2.34.0
> jersey2-hk2.jar updated from 2.29.1 to 2.34.0
> jersey2-server.jar updated from 2.29.1 to 2.34.0
