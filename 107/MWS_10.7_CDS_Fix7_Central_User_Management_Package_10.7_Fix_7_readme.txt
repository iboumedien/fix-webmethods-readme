6.0 Cautions and Warnings

If this is the first Common Directory Services fix that you are installing, you must perform
the following steps to enable calendar administration:
1. In a text editor, copy and paste the following xml snippet on a single line:
<?xml version="1.0" encoding="UTF-8"?><CONFIG><executecommand commandName="installComponent"
componentLocation="class:/is_components/wm_calendar_admin.war" resourceURI=""/></CONFIG>
2. In Inetgration Server, go to Packages > Management > WmCDS > Browse services in WmCDS and
locate the wm.cds.services_java:importXml service.
3. Provide the line from step 1. as input to the service and click Test (with inputs).
