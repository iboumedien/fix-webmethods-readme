7.0 Fix Contents

In addition to including resolutions to product defects, a fix may also include enhancements to 
existing functionality and features. Any of these items can result in new or changed built-in 
services,
APIs, or configuration parameters. Review the fix contents carefully before installing the fix.

This fix is cumulative, which means that it includes all previous fixes 
for the release.

WOK-15456 (WOK_10.7_RMC_Fix3)
The value of the condition cell is not defined. [RUL.0070.0970]
When using the service PUT 
%MWS%/wm_rma/rest/content/project/AflacInvoiceReconRules/decisiontable/%decisiontable%, the server 
returns an "The value of the condition cell is not defined. [RUL.0070.0970]" error, when the caller 
is trying to replace an empty cell with a cell containing an operator and a value.
This issue is now resolved.

WOK-15570 (WOK_10.7_RMC_Fix8)
Error displaying business rules in MWS
When clicking on the navigation item "Administration - Business - webMethods Business Rules - rule 
project - Decision Tables" in the left-hand side navigation bar, sometimes the Decision Table 
portlets on the right are not displayed and an empty page is shown.
This issue has been resolved by refactoring the Business Rules navigation bar.

WOK-15663 (WOK_10.7_RMC_Fix4)
NullPointerException during rule project builds
When building a rule project with the Asset Build Environment, the build would fail with
java.lang.NullPointerException
at com.wm.lang.ns.NSRecordRef.getNSRecord(NSRecordRef.java:245)
This issue is now resolved.

WOK-16308 (WOK_10.7_RMC_Fix8)
Unsynchronized navigation entries with concurrent users
In rare occasions, there could be wrong Business Rules navigation entries when multiple concurrent 
users are logging in.
This issue has been resolved.

WOK-16333 (WOK_10.7_RMC_Fix6)
Jackson library v2.12.3 causes com.fasterxml.jackson.databind.exc.InvalidDefinitionException
When using Jackson library version 2.12.3, reading Business Rules auditing information from the 
database results in a com.fasterxml.jackson.databind.exc.InvalidDefinitionException.
This issue has been resolved.

WOK-16486 (WOK_10.7_RMC_Fix8)
Business Rules does not support MySQL 8.x CE for 10.7
Business Rules 10.7 did not support MySQL 8,x Community Edition.
This issue is now resolved.

WOK-16537 (WOK_10.7_RMC_Fix9)
Edit function is clearing parameter in functions editor
When using the graphical function editor with a combination of parameters and functions, the 
parameter is incorrectly deleted.
This issue has been resolved.

WOK-16542 (WOK_10.7_RMC_Fix9)
Enable graphical function editor for editing functions
When editing functions in the Rules Management Console, one can not use the the same graphical 
editor which can be used for creating functions.
This issue has been resolved.


WOK-16577 (WOK_10.7_RMC_Fix9)
Pencil in expression builder dialog stays inactive on IE
When using Internet Explorer to open the graphical function editor, the corresponding icon is not 
enabled.
This issue has been resolved.

WOK-17437
Security issue within sag-web-components.js
sag-web-components.js contained outdated and potentially vulnerable libraries moment.js and jquery.
This issue is resolved by updating these libraries to moment.js 2.29.3 and jquery 3.6.0.
